# Python 常用内置函数速查

> 内置函数 = 不需要 `import`，直接就能用的函数

---

## 一、输入输出

```python
print("hello")           # 输出到控制台
print(1, 2, 3, sep="-")  # 1-2-3，sep 指定分隔符
print("a", end="")       # 不换行，默认 end="\n"

input("请输入：")          # 从控制台读入，返回字符串
name = input("姓名：")     # 结果是 str 类型，需要数字要自己转
```

---

## 二、类型转换

```python
int("42")       # 字符串 → 整数：42
int(3.9)        # 浮点数 → 整数：3（截断，不是四舍五入）
float("3.14")   # 字符串 → 浮点数：3.14
str(100)        # 任意类型 → 字符串："100"
bool(1)         # → True；bool(0) → False；bool("") → False
list("abc")     # → ['a', 'b', 'c']
tuple([1,2,3])  # → (1, 2, 3)
set([1,2,2,3])  # → {1, 2, 3}（去重）
dict([("a",1)]) # → {'a': 1}
```

---

## 三、序列操作

```python
len([1, 2, 3])        # 长度：3
len("你好")            # 字符串长度：2（一个中文字算一个）

range(5)              # → 0,1,2,3,4
range(2, 5)           # → 2,3,4
range(0, 10, 2)       # → 0,2,4,6,8

enumerate(["a","b"])  # → (0,"a"), (1,"b")，遍历时带索引
zip([1,2], ["a","b"]) # → (1,"a"), (2,"b")，并行遍历多个列表

reversed([1,2,3])     # → 3,2,1（返回迭代器）
sorted([3,1,2])       # → [1,2,3]（返回新列表）
sorted([3,1,2], reverse=True)  # → [3,2,1]
```

---

## 四、数学相关

```python
abs(-5)          # 绝对值：5
max(1, 3, 2)     # 最大值：3
max([1, 3, 2])   # 也支持列表
min(1, 3, 2)     # 最小值：1
sum([1, 2, 3])   # 求和：6
round(3.14159)   # 四舍五入：3
round(3.14159, 2) # 保留2位小数：3.14

pow(2, 3)        # 2³：8，等价于 2**3
divmod(10, 3)    # (商, 余数)：(3, 1)
```

---

## 五、类型判断与检查

```python
type(123)        # <class 'int'>
type("abc")      # <class 'str'>
type([1,2])      # <class 'list'>

isinstance(123, int)        # True
isinstance("abc", str)      # True
isinstance([1,2], (list, tuple))  # 检查多种类型

id(123)          # 对象的内存地址（整数）
id("hello")      # 字符串的内存地址
```

---

## 六、其他常用

```python
# 查看对象的所有属性和方法
dir([])          # 列出列表的所有方法

# 返回布尔值
all([True, 1, "a"])    # True（所有元素为真）
any([0, "", "a"])      # True（至少一个为真）

# 帮助
help(print)      # 查看函数的文档
```
