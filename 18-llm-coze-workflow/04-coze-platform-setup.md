# 04. 扣子平台准备

> 前置：[[01-ai-trend-and-coze-basic]]
> 目标：完成注册、创建空间、理解免费版限制
> 用时：约 5 分钟（主要是实操）
>
> 相关笔记：[[05-workflow-entry-practice]]

---

## 一、账号注册与工作空间

### 步骤

1. 访问扣子官网（coze.cn 或 byteplus 扣子入口）
2. 注册账号（手机号/邮箱）
3. 登录控制台
4. 创建个人工作空间（Space）
5. 在工作空间中创建项目（Project）

### 空间的概念

一个空间 = 一个独立的工作环境。不同空间之间的资源（插件、知识库、工作流）隔离。

建议：
- 学习和实验 → 一个空间
- 实际项目 → 每个项目一个空间

---

## 二、免费版与专业版差异

| 对比项 | 免费版 | 专业版 |
|--------|--------|--------|
| 每日模型调用 | 有限额（通常几百次） | 更高额度 |
| 可用模型 | GPT-4o mini、豆包等轻量模型 | GPT-4、Claude 等高级模型 |
| 插件数量 | 基础插件可用 | 更多插件 |
| 工作流节点数 | 有限制 | 更宽松 |
| 知识库容量 | 有限 | 更大 |
| 商业发布 | 受限 | 支持 |
| 团队协作 | 不支持 | 支持 |

**学习阶段免费版完全够用。** 如果额度不够，换一个模型或等第二天重置。

---

## 三、插件与知识库权限

### 插件授权

某些插件（如天气、地图）可能需要：
- 在空间设置中开启
- 同意第三方服务条款
- 配置 API Key（少数高级插件）

### 知识库发布

创建知识库后需要**发布**才能被工作流/智能体调用：
1. 上传文档（PDF、Word、TXT、Markdown）
2. 等待处理和索引
3. 点击"发布"
4. 在智能体或工作流中关联知识库

---

## 四、智能体 vs 工作流 vs 独立工作流

| 形式 | 特点 | 适用场景 |
|------|------|---------|
| **智能体** | 有人设、会对话、自动理解意图 | 客服、助手、问答 |
| **智能体内的工作流** | 被智能体调用，处理特定任务 | 对话中触发某个流程 |
| **独立工作流** | 不依赖对话，直接运行 | 批量处理、定时任务、API 调用 |

**学习建议**：先从独立工作流开始，因为它结构最清晰（输入→处理→输出），不受对话上下文干扰。

---

## 五、你现在就可以做的

1. 打开浏览器，注册扣子账号
2. 创建空间，命名为"学习实验"
3. 随意探索一下界面：看看有哪些插件、模板、知识库
4. 不用现在就搭工作流——下一讲带着你做
