# 08. 行业案例

> 前置：[[07-workflow-advanced-integration]]

>
> 相关笔记：[[09-workflow-projects]]

---

## 一、自媒体场景

### 可做项目

| 项目 | 工作流节点 | 产出 |
|------|-----------|------|
| 图文生成 | 开始 → LLM(写文案) → 回复 | 可直接发布的小红书/公众号图文 |
| 标题生成 | 开始 → LLM(生成N个标题) → 回复 | 10 个备选标题 |
| 封面文案 | 开始 → LLM(写文案) → 插件(生成图) → 回复 | 封面图 + 文案 |
| 视频脚本 | 开始 → LLM(脚本) → 代码块(格式化) → 回复 | 分镜脚本 |

### 核心价值

**批量 + 模板化**。一个人一天最多写 5 篇原创图文，扣子工作流可以 1 分钟出 10 个选题 + 10 篇初稿。

## 三、智能客服

| 类型 | 实现方式 |
|------|---------|
| 售前客服 | 知识库（产品信息）+ LLM（自然回答） |
| 售后客服 | 知识库（退换货政策）+ 条件分支（判断问题类型） |
| FAQ 问答 | 知识库直接检索 + LLM 润色 |
| 工单分类 | LLM 判断意图 → 条件分支 → 分派到不同处理流程 |

## 五、行业批量出图

适合需要大量视觉内容的行业：

| 行业 | 典型工作流 |
|------|-----------|
| 装修 | 输入户型 → 生成装修效果图 + 解说文案 |
| 牙医 | 输入症状 → 生成治疗前后对比图 + 科普文案 |
| 旅游 | 输入目的地 → 生成景点宣传图 + 攻略 |
| 美妆 | 输入产品 → 生成试色图 + 种草文案 |
| 教育 | 输入知识点 → 生成教学卡片 + 讲解视频脚本 |
| 美食 | 输入菜名 → 生成美食图 + 食谱卡片 |

### 通用批量出图工作流

```
输入：行业 + 主题列表
    ↓
循环节点（对每个主题）：
    LLM 生成图片 prompt
    → 图片生成插件（异步）
    → LLM 生成配套文案
    ↓
回复：N 张图 + N 段文案
```

## 六、你的切入点

`★ Insight ─────────────────────────────────────`
- 你是软工背景，和纯粹的业务人员比，你的优势在于**能写代码块节点 + 能调 HTTP 插件对接后端**。纯 LLM 节点大家都能配，但代码块和数据对接是你的护城河。
- 选方向时优先选**你了解领域的**。比如你做过苍穹外卖，对餐饮/外卖的业务逻辑有感知，那"餐饮行业 AI 工作流"就是一个很自然的切入点。
- 这些行业案例学完后，挑 2 个实际做出来——一个自媒体类的（好展示），一个你专业的（好深入）。后续简历上可以写"基于 Coze 平台独立搭建了 X 和 Y 两个 AI 工作流"。
`─────────────────────────────────────────────────`
