## 三、改错题

每题按“错误现象 → 错误原因 → 最小修改”来做。

### 题 1：split 后忘记转整数

```python
scores = "80 90 70".split()
print(sum(scores))
```

正确：

```python
scores = []
for part in "80 90 70".split():
    scores.append(int(part))
print(sum(scores))
```

### 题 2：join 连接整数列表

```python
nums = [1, 2, 3]
print(",".join(nums))
```

正确：

```python
nums = [1, 2, 3]
parts = []
for num in nums:
    parts.append(str(num))
print(",".join(parts))
```

### 题 3：字典直接与整数相加

```python
student = {"name": "Tom", "score": 80}
print(student + 10)
```

正确：

```python
print(student["score"] + 10)
```

### 题 4：推导式判断错变量

```python
n = 10
result = [i for i in range(1, n + 1) if n % 2 == 0]
```

正确：

```python
result = [i for i in range(1, n + 1) if i % 2 == 0]
```

### 题 5：循环时删除原列表

```python
nums = [1, 2, 3, 4]
for num in nums:
    if num % 2 == 0:
        nums.remove(num)
```

正确：

```python
result = []
for num in nums:
    if num % 2 != 0:
        result.append(num)
```

### 题 6：忘记 return

```python
def add(a, b):
    result = a + b

print(add(1, 2))
```

正确：

```python
def add(a, b):
    return a + b
```

### 题 7：默认参数使用空列表

```python
def add_student(name, students=[]):
    students.append(name)
    return students
```

正确：

```python
def add_student(name, students=None):
    if students is None:
        students = []
    students.append(name)
    return students
```

### 题 8：平均值在循环中计算

```python
total = 0
for score in scores:
    total += score
    average = total / len(scores)
```

正确：

```python
total = 0
for score in scores:
    total += score
average = total / len(scores)
```

### 题 9：使用累计总分比较最高分

```python
highest = scores[0]
total = 0
for score in scores:
    total += score
    if total > highest:
        highest = total
```

正确：

```python
highest = scores[0]
for score in scores:
    if score > highest:
        highest = score
```

### 题 10：`**student` 键与参数名不一致

```python
def show_student(name, score):
    print(name, score)

student = {"username": "Tom", "score": 80}
show_student(**student)
```

正确：

```python
student = {"name": "Tom", "score": 80}
show_student(**student)
```

### 题 4：两个列表生成字典

普通循环：

```python
names = ["Tom", "Jerry"]
scores = [80, 92]
result = {}
for name, score in zip(names, scores):
    result[name] = score
```

`dict(zip())`：

```python
result = dict(zip(names, scores))
```

字典推导式：

```python
result = {name: score for name, score in zip(names, scores)}
```

### 题 6：按成绩排序

元组列表：

```python
students = [("Tom", 80), ("Jerry", 92)]

```

字典列表：

```python
students = [{"name": "Tom", "score": 80}, {"name": "Jerry", "score": 92}]
```

## 五、最终综合题：学生成绩管理

### 输入要求

输入学生人数，然后输入：

```text
姓名,成绩
```

例如：

```text
Tom,80
Jerry,92
Alice,55
```

### 功能要求

1. 使用 `strip()` 清理输入。
2. 使用 `split(",")` 拆分。
3. 将成绩转为整数。
4. 每个学生保存为字典。
5. 所有学生保存到列表。
6. 输出最高分学生。
7. 输出最低分学生。
8. 输出平均分。
9. 输出及格人数。
10. 返回不及格学生列表。
11. 按成绩降序排列。
12. 成绩相同时按姓名排序。
13. 使用 `show_student(name, score)`。
14. 至少一次通过 `**student` 调用。
15. 使用 `join()` 生成排名文本。
16. 不使用全局变量。

### A. 基础循环版

```python
def parse_student(line):
    name, score_text = line.strip().split(",")
    return {"name": name, "score": int(score_text)}


def input_students():
    count = int(input("请输入学生人数："))
    students = []

    for i in range(count):
        line = input("请输入姓名,成绩：")
        student = parse_student(line)
        students.append(student)

    return students


def analyze_students(students):
    top = students[0]
    low = students[0]
    total = 0
    passed = 0

    for student in students:
        score = student["score"]
        total += score

        if score > top["score"]:
            top = student

        if score < low["score"]:
            low = student

        if score >= 60:
            passed += 1

    average = total / len(students)

    return {
        "top": top,
        "low": low,
        "average": average,
        "passed": passed,
    }


def get_failed_students(students):
    failed = []
    for student in students:
        if student["score"] < 60:
            failed.append(student)
    return failed


def sort_students(students):
    return sorted(students, key=lambda student: (-student["score"], student["name"]))


def show_student(name, score):
    return f"{name}:{score}"


def format_students(students):
    lines = []
    for student in students:
        lines.append(show_student(**student))
    return "\n".join(lines)


def main():
    students = input_students()
    result = analyze_students(students)
    failed = get_failed_students(students)
    sorted_students = sort_students(students)

    print("最高分：", show_student(**result["top"]))
    print("最低分：", show_student(**result["low"]))
    print("平均分：", result["average"])
    print("及格人数：", result["passed"])
    print("不及格学生：", failed)
    print("排名：")
    print(format_students(sorted_students))


if __name__ == "__main__":
    main()
```

### B. 简化提升版

考试中不会简化写法时，优先写基础循环版。

```python
def parse_student(line):
    name, score_text = line.strip().split(",")
    return {"name": name, "score": int(score_text)}


def analyze_students(students):
    return {
        "top": max(students, key=lambda student: student["score"]),
        "low": min(students, key=lambda student: student["score"]),
        "average": sum(student["score"] for student in students) / len(students),
        "passed": sum(1 for student in students if student["score"] >= 60),
    }


def get_failed_students(students):
    return [student for student in students if student["score"] < 60]


def sort_students(students):
    return sorted(students, key=lambda student: (-student["score"], student["name"]))


def show_student(name, score):
    return f"{name}:{score}"


def format_students(students):
    return "\n".join(show_student(**student) for student in students)
```
