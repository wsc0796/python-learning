# Day 1 程序阅读题

先写预测输出，再运行验证。不要先运行。

## 题 1（考点：类型转换，建议 3 分钟）

```python
a = "3"
b = 4
print(a * 2)
print(int(a) + b)
```

你的预测：33 7

## 题 2（考点：range，建议 4 分钟）

```python
for i in range(2, 9, 3):
    print(i, end=" ")
```

你的预测： 2 5 8

## 题 3（考点：continue，建议 5 分钟）

```python
total = 0
for i in range(1, 6):
    if i == 3:
        continue
    total += i
print(total)
``` 


你的预测： 12

## 题 4（考点：break 和循环 else，建议 6 分钟）

```python
for i in range(1, 5):
    if i == 6:
        break
else:
    print("done")
```

你的预测：done

## 题 5（考点：while，建议 6 分钟）

```python
n = 5
while n > 0:
    n -= 2
    print(n)
```

你的预测：3,1,-1
