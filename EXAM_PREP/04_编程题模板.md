# 编程题模板

## 1. 循环统计

```python
total = 0
count = 0
for x in 数据:
    if 条件:
        total += x
        count += 1
print(total, count)
```

## 2. 列表筛选

```python
result = []
for item in items:
    if 条件:
        result.append(item)
print(result)
```

## 3. 字典计数

```python
counts = {}
for ch in text:
    counts[ch] = counts.get(ch, 0) + 1
print(counts)
```

## 4. 函数拆分

```python
def 函数名(参数):
    # 处理过程
    return 结果
```

## 5. 类模板

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def show(self):
        print(self.name, self.score)
```

## 6. 文件读取

```python
with open("data.txt", "r", encoding="utf-8") as f:
    for line in f:
        line = line.strip()
        # 处理 line
```

## 7. 文件写入

```python
with open("result.txt", "w", encoding="utf-8") as f:
    f.write("内容\n")
```

## 8. 异常处理

```python
try:
    value = int(input("请输入整数："))
except ValueError:
    print("输入错误")
else:
    print(value)
finally:
    print("程序结束")
```

## 9. 菜单循环

```python
while True:
    print("1. 添加")
    print("2. 查询")
    print("0. 退出")
    choice = input("请选择：")

    if choice == "1":
        pass
    elif choice == "2":
        pass
    elif choice == "0":
        break
    else:
        print("输入无效")
```
