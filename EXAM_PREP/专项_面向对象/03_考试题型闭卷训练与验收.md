```python
class Demo:
```

### 题 8：继承

```python
class Animal:
    def run(self):
        print("run")

class Dog(Animal):
```

### 题 9：方法重写

```python
class Animal:
    def cry(self):
        print("...")

class Dog(Animal):

Dog().cry()
```

### 题 10：super

```python
class User:
    def show(self):
        return "User"

class Admin(User):
    def show(self):
        return super().show() + "+Admin"

print(Admin().show())
```
User+Admin
### 题 11：多态

```python
class Dog:
    def speak(self):
        return "汪"

class Cat:
    def speak(self):
        return "喵"

def f(animal):
    print(animal.speak())

f(Dog())
f(Cat())
```
汪，喵
### 题 12：多继承顺序

```python
class A:
    def show(self):
        return "A"

class B:
    def show(self):
        return "B"

class C(A, B):
    pass

print(C().show())
```

### 题 13：`__add__`

```python
class Score:
    def __init__(self, value):
        self.value = value

    def __add__(self, other):
        return Score(self.value + other.value)

print((Score(80) + Score(10)).value)
```
90

```python
d = Demo()
del d

```

### 题 15：对象列表

## 二、改错题

### 题 1：方法漏写 self

错误：

```python
class Student:
    def show():
        print("hello")
```

正确：

```python
class Student:
    def show(self):
        print("hello")
```

### 题 2：`self.name` 写成 `name`

```python
class Student:
    def __init__(self, name):
        name = name
```

最小修改：`self.name = name`。

### 题 3：创建对象参数数量错误

```python
class Car:
    def __init__(self, color):
        self.color = color

car = Car()
```

最小修改：`car = Car("red")`。

### 题 4：`__init__` 拼写错误

```python
class A:
    def init(self):
        self.name = "A"
```

最小修改：`def __init__(self):`。

### 题 5：`__str__` 没有返回字符串

```python
def __str__(self):
    print("hello")
```

最小修改：`return "hello"`。

### 题 6：子类漏调父类初始化

```python
class Admin(User):
    def __init__(self, authority):
        self.authority = authority
```

最小修改：补 `super().__init__(...)`。

### 题 7：super 调用语法错误

```python
super.__init__(name)
```

正确：`super().__init__(name)`。

### 题 8：外部直接访问私有属性

```python
print(user.__password)
```

正确：`print(user.get_password())`。

### 题 9：类属性与实例属性混淆

通过对象赋值同名属性不会修改所有对象的类属性。

### 题 10：`__add__` 没有返回结果

```python
def __add__(self, other):
    Score(self.value + other.value)
```

正确：`return Score(...)`。

### 题 11：静态方法错误使用 self

静态方法没有自动传入 `self`，需要参数就显式写。

### 题 12：依赖 `__del__` 保存重要数据

资源清理和文件保存优先用 `with` 或显式方法，不依赖析构。

### B. 结构优化版

```python
class User:
    def __init__(self, name):
        self.name = name

    def show_info(self):
        return self.name


class Admin(User):
    def __init__(self, name, authority):
        super().__init__(name)
        self.authority = authority

    def show_info(self):
        return f"{super().show_info()}:{self.authority}"
```
