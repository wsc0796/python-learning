## self【必须闭卷会写】

### 1. 它解决什么问题

`self` 表示“当前这个对象自己”。同一个类可以创建很多对象，方法里必须知道现在操作的是哪一个对象。

### 2. 最小语法

```python
class Student:
    def show(self):
        print(self)


student = Student()
student.show()
```

### 3. 对象创建和调用过程

```text
student.show()
→ Python 自动把 student 传给 self
→ show(self) 里的 self 就是 student
```

### 4. 常见错误

```python
class Student:
    def show():
        print("hello")


Student().show()
```

错误原因：实例方法漏写 `self`，对象调用时 Python 会自动传一个对象进去，参数数量对不上。

### 5. 考试可能怎么出

改错题：方法定义漏写 `self`。

## 构造方法 `__init__`【必须闭卷会写】

### 1. 它解决什么问题

创建对象时，把必要数据一次性保存到对象里。

### 2. 最小语法

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score


student = Student("Tom", 80)
print(student.name)
print(student.score)
```

### 3. 对象创建和调用过程

```text
Student("Tom", 80)
→ 自动调用 __init__(self, "Tom", 80)
→ self.name = "Tom"
→ self.score = 80
→ 数据保存在对象里
```

`name` 和 `self.name` 的区别：

```text
name：形参，只是 __init__ 里的临时变量
self.name：实例属性，保存在对象中
```

### 4. 常见错误

```python
class Student:
    def __init__(self, name):
        name = name
```

错误原因：左边的 `name` 还是局部变量，没有写成 `self.name`，对象里没有保存属性。

### 5. 考试可能怎么出

要求创建对象时传入属性，例如 `Circle(0, 0, 5, "red")`。

		self.color=color

## 三种方法【必须会读会改】

### 1. 它解决什么问题

不同方法接收的“第一个东西”不一样：对象、类，或者什么都不自动接收。

### 2. 最小语法

```python
class Example:
    count = 0

    def instance_method(self):
        return "实例方法"

    @classmethod
    def class_method(cls):
        return cls.count

    @staticmethod
    def static_method(score):
        return score >= 60
```

### 3. 对象创建和调用过程

```text
self 接收当前对象
cls 接收当前类
staticmethod 不自动接收对象或类
```

### 4. 常见错误

在静态方法里直接使用 `self`：

```python
@staticmethod
def is_pass():
    return self.score >= 60
```

静态方法没有 `self`。

### 5. 考试可能怎么出

阅读题：判断实例方法、类方法、静态方法分别能访问什么。

|方法类型|能访问实例属性？|能访问类属性？|原因|
|---|:-:|:-:|---|
|实例方法 `(self)`|✅ `self.name`|✅ `self.total` 或 `类名.total`|self 连接着对象|
|类方法 `(cls)`|❌|✅ `cls.total`|cls 连接着类，没连对象|
|静态方法 `()`|❌|❌（除非写 `类名.xxx`）|什么都没连|
### 6. 最小闭卷题

写一个 `ScoreTool`，静态方法判断成绩是否及格。

```python


class ScoreTool:
	@staticmethod
	def is_pass(score):
		if score>= 60：
			return true
		else： return false
	
	
	
print(ScoreTool.is_pass(80))
print(ScoreTool.is_pass(50))
```

```python
class Student:

        self.score = score

    def __str__(self):

student = Student("Tom", 80)

```

```text
print(student)
```

```python
def __str__(self):
```

## 私有属性与课程类【必须闭卷会写】

### 1. 它解决什么问题

实验报告题：课程类包含课程名、编号、老师、上课地点，其中地点是私有属性。

### 2. 最小语法

```python
class Course:
    def __init__(self, name, number, teacher, location):
        self.name = name
        self.number = number
        self.teacher = teacher
        self.__location = location

    # ── get：读取 ──
    def get_location(self):
        return self.__location

    # ── set：修改 ──
    def set_location(self, location):
        self.__location = location

    def set_teacher(self, teacher):
        self.teacher = teacher

    def show_info(self):
        print(f"课程名：{self.name}")
        print(f"课程编号：{self.number}")
        print(f"任课老师：{self.teacher}")
        print(f"上课地点：{self.__location}")
```

### 3. 对象创建和调用过程

```python
c = Course("Python", "PY001", "王老师", "A101")

# get — 读
print(c.get_location())           # A101

# set — 改
c.set_location("B203")
c.set_teacher("李老师")

# 改了以后再查
print(c.get_location())           # B203
c.show_info()
# 输出：课程名：Python  ...  任课老师：李老师  上课地点：B203
```

```text
Course(...)
→ __location 被保存为私有属性
→ 外部不要直接 course.__location
→ 读：get_location()      写：set_location()
```

### 4. 常见错误

```python
course = Course("Python", "PY001", "王老师", "A101")
print(course.__location)
```

会报 `AttributeError`。双下划线会发生名称改写，不是绝对安全，但外部应通过方法访问。

### 5. `@property` 最小版本【必须会读会改】

```python
class Course:
    def __init__(self, name, location):
        self.name = name
        self.__location = location

    @property
    def location(self):
        return self.__location


course = Course("Python", "A101")
print(course.location)
```

### 6. 最小闭卷题

定义 `Course` 类，`location` 私有，通过 `get_location()` 返回地点。

### 7. get/set 方法 ↔ @property 对照

```python
class Course:
    def __init__(self, name, location):
        self.name = name
        self.__location = location

    # ── 方式 A：get / set 普通方法 ──
    def get_location(self):
        return self.__location

    def set_location(self, value):
        self.__location = value

    # ── 方式 B：@property（效果完全一样）──
    @property
    def location(self):
        return self.__location

    @location.setter
    def location(self, value):
        self.__location = value
```

```text
调用对比（功能一样，写法不同）：
                 方式 A                   方式 B
  读     course.get_location()       course.location
  写     course.set_location("B")    course.location = "B"
  校验   写在 set_location 里         写在 @location.setter 里
```

| | 普通方法 | @property |
|------|------|------|
| 读 | `obj.get_xxx()` | `obj.xxx` |
| 写 | `obj.set_xxx(v)` | `obj.xxx = v` |
| 括号 | 必须写 | 不用写 |
| 好处 | 跟你 Java 习惯一致 | 更简洁，后面加校验外面不用改 |

## 对象列表【必须会读会改】

### 1. 它解决什么问题

多个对象通常放进列表，例如多个学生。

### 2. 最小语法

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def show(self):
        return f"{self.name}:{self.score}"


students = [
    Student("A", 80),
    Student("B", 90),
]

for student in students:
    print(student.show())

top = max(students, key=lambda student: student.score)
print(top.name)
```

### 3. 常见错误

```python
top = max(students)
```

对象默认不知道按什么比较，要写 `key=lambda student: student.score`。

## 属性与方法对照表

| 内容 | 写法 | 保存在哪里 | 作用 |
|---|---|---|---|
| 实例属性 | `self.name` | 对象 | 保存对象状态 |
| 类属性 | `Student.school` | 类 | 所有对象共享 |
| 实例方法 | `def show(self)` | 类中定义，对象调用 | 操作对象 |
| 类方法 | `@classmethod` | 类中定义 | 操作类 |
| 静态方法 | `@staticmethod` | 类中定义 | 工具函数 |

## 10 个易错点

1. 创建对象要写 `类名()`。
2. 实例方法第一个参数必须写 `self`。
3. `self.name = name` 才是保存到对象。
4. `__init__` 前后都是双下划线。
5. 创建对象传参数量要和 `__init__` 匹配。
6. 类属性可能被对象同名属性遮蔽。
7. `__str__` 必须返回字符串。
8. 私有属性不能直接用 `obj.__attr` 访问。
9. 静态方法没有 `self`。
10. 对象列表排序要指定 `key`。

```python
        self.score = score

    def __str__(self):
        return f"{self.name}: {self.score}分"


students = [
    Student("小明", 80),
    Student("小红", 90),
    Student("小刚", 75),
]

# 方式 A：max + key（一行）
best = max(students, key=lambda s: s.score)
print(f"最高分：{best}")                # 最高分：小红: 90分

# 方式 B：for 循环（逻辑更直白）
best = students[0]
for s in students:
    if s.score > best.score:
        best = s
print(f"最高分：{best}")                # 最高分：小红: 90分
```
