```text

5. else 是否执行？
```

## 一、程序输出题

### 题 1：`try/except/finally`

```python
try:
    n = int("abc")
except ValueError:
    print("error")
else:
    print("ok")
finally:
    print("end")
```

### 题 2：`else`

```python
try:
    n = int("12")
except ValueError:
    print("error")
else:
    print(n)
```

### 题 3：`finally`

```python
try:
    print(1)
finally:
    print(2)
```

### 题 4：`return + finally`

```python
def f():
    try:
        return 1
    finally:
        print("finally")

print(f())
```

### 题 5：异常传递

```python
def parse():
    return int("abc")

try:
    parse()
except ValueError:
    print("bad")
```

### 题 6：raise

```python
def check(n):
    if n < 0:
        raise ValueError("不能为负")
    print(n)

check(3)
```

### 题 7：自定义异常

```python
class MyError(Exception):
    pass

try:
    raise MyError("wrong")
except MyError as error:
    print(error)
```

### 题 8：多个 except

```python
try:
    result = 10 / 0
except ValueError:
    print("value")
except ZeroDivisionError:
    print("zero")
```

### 题 9：文件模式 `w`

```python
with open("a.txt", "w", encoding="utf-8") as file:
    file.write("hello")
```

说明：已有 `a.txt` 会被覆盖。

### 题 10：文件指针

```python
with open("a.txt", "r", encoding="utf-8") as file:
    text1 = file.read()
    text2 = file.read()
```

说明：第二次通常为空。

### 题 11：`readlines`

`readlines()` 返回字符串列表。

### 题 12：`with open`

`with` 结束时文件自动关闭。

7. `wrong`

## 二、改错题

### 题 1：`r` 打开不存在文件

应捕获 `FileNotFoundError`。

```python
try:
    with open("none.txt", "r", encoding="utf-8") as file:
        print(file.read())
except FileNotFoundError:
    print("文件不存在")
```

### 题 2：`w` 模式误覆盖

如果要保留旧内容，用 `a`。

### 题 3：忘记编码

中文文本建议写 `encoding="utf-8"`。

### 题 4：`write()` 传入整数

```python
file.write(str(123))
```

### 题 5：`writelines()` 不自动换行

```python
file.writelines(["a\n", "b\n"])
```

### 题 6：捕获错误异常类型

`int("abc")` 应捕获 `ValueError`。

### 题 7：裸 `except`

不推荐：

```python
except:
    ...
```

推荐捕获明确异常。

### 题 8：`try` 放太多代码

只把真正可能抛异常的代码放进去。

### 题 9：`finally` 中访问未创建文件对象

用 `with open` 避免。

### 题 10：`raise` 后仍写正常逻辑

`raise` 后流程中断，后续正常逻辑不会执行。

### 题 11：自定义异常未继承 `Exception`

```python
class MyError(Exception):
    pass
```

### 题 12：只用 `isdigit()` 处理所有数字

`-1`、`3.14` 都会让 `isdigit()` 为 False，整数转换仍建议 `try/except`。

```python
def safe_divide(a, b):
```

```python
def read_file(path):
```

```python
    with open(source, "r", encoding="utf-8") as src:

    with open(target, "w", encoding="utf-8") as dst:
```

## 四、最终综合题：学生成绩文件处理系统

输入文件格式：

```text
Tom,80
Jerry,abc
Alice,105
Bob,60
```

要求：

```text
with open
逐行读取
strip 和 split(",")
合法成绩转整数
非整数捕获 ValueError
超范围抛自定义异常
错误行记录到列表
合法学生保存为字典列表
计算最高分和平均分
结果与错误信息分别写文件
文件不存在时提示
finally 输出处理结束
不使用全局变量
```

### A. 基础保底版

```python
class InvalidScoreError(Exception):
    pass


def parse_student_line(line):
    name, score_text = line.strip().split(",")
    score = int(score_text)

    if not 0 <= score <= 100:
        raise InvalidScoreError("成绩必须在 0 到 100 之间")

    return {"name": name, "score": score}


def read_students(path):
    students = []
    errors = []

    try:
        with open(path, "r", encoding="utf-8") as file:
            for line_number, line in enumerate(file, start=1):
                try:
                    student = parse_student_line(line)
                except ValueError:
                    errors.append(f"第{line_number}行：成绩不是整数")
                except InvalidScoreError as error:
                    errors.append(f"第{line_number}行：{error}")
                else:
                    students.append(student)
    except FileNotFoundError:
        print("文件不存在")
    finally:
        print("处理结束")

    return students, errors
```

### B. 函数拆分完整版本

```python
def analyze_students(students):
    top = max(students, key=lambda student: student["score"])
    total = 0

    for student in students:
        total += student["score"]

    return {
        "top": top,
        "average": total / len(students),
    }


def write_report(path, result):
    with open(path, "w", encoding="utf-8") as file:
        file.write(f'最高分：{result["top"]["name"]},{result["top"]["score"]}\n')
        file.write(f'平均分：{result["average"]:.2f}\n')


def write_errors(path, errors):
    with open(path, "w", encoding="utf-8") as file:
        for error in errors:
            file.write(error + "\n")
```
