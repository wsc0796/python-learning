# OOP 实验报告程序

## 文件说明

- `circle_demo.py`：圆类 Circle，测试圆心、半径、颜色、周长、面积。
- `course_demo.py`：课程类 Course，`location` 为私有属性。
- `friend_manager.py`：好友关系系统，包含添加、删除、备注、展示、分组、退出。
- `mysystem.py`：用户类 User 和管理员类 Admin。
- `test.py`：测试 `mysystem.py` 中的 User 和 Admin。

## 运行命令

```bash
python EXAM_PREP/lab_oop_report/circle_demo.py
python EXAM_PREP/lab_oop_report/course_demo.py
python EXAM_PREP/lab_oop_report/friend_manager.py
python EXAM_PREP/lab_oop_report/test.py
```

## 注意

题目里写的是 `Cirle`，这里按英文正确拼写使用 `Circle`。
