# 高频语法速记

## 基础语法

- `input()` 得到的一定是字符串，需要 `int()` 或 `float()` 转换。
- `//` 是整除，`%` 是取余，`**` 是幂。
- 比较运算结果是 `True` 或 `False`。
- `and` 两边都真才真，`or` 一边真就真，`not` 取反。

## 流程控制

```python
if 条件:
    ...
elif 条件:
    ...
else:
    ...
```

```python
for i in range(开始, 结束, 步长):
    ...
```

- `range(1, 5)` 产生 1、2、3、4，不包含 5。
- `break` 结束整个循环。
- `continue` 跳过本轮，进入下一轮。
- 循环 `else`：循环没有被 `break` 打断时才执行。

## 字符串

- `s[i]` 取单个字符。
- `s[a:b]` 左闭右开。
- `find()` 找不到返回 `-1`。
- `replace()` 返回新字符串，不会原地修改。
- `split()` 字符串转列表。
- `join()` 列表拼回字符串。

## 组合数据类型

- 列表可变：`append`、`insert`、`remove`、`pop`、`sort`。
- 元组不可变：常用于固定数据。
- 字典按 key 取值：`d[key]` 不存在会报错，`d.get(key, 默认值)` 更安全。
- 集合自动去重：`set(list_data)`。

## 函数

```python
def 函数名(参数):
    return 返回值
```

- 没有 `return` 默认返回 `None`。
- 默认参数放在普通参数后面。
- `*args` 接收多余位置参数，结果是元组。
- `**kwargs` 接收多余关键字参数，结果是字典。
- `global` 修改全局变量。
- `nonlocal` 修改外层函数变量。

## 面向对象

```python
class Student:
    def __init__(self, name):
        self.name = name

    def show(self):
        print(self.name)
```

- `self` 代表当前对象。
- `类名()` 创建对象。
- 实例属性属于对象，类属性属于类。
- 子类找方法：先找自己，再找父类。
- 私有属性通常写成 `__name`。

## 文件与异常

```python
with open("a.txt", "r", encoding="utf-8") as f:
    content = f.read()
```

- `r` 读，文件不存在会报错。
- `w` 写，会覆盖原文件，不存在会创建。
- `a` 追加，不存在会创建。
- `try` 放可能出错的代码。
- `except` 处理异常。
- `else` 没异常时执行。
- `finally` 无论是否异常都会执行。
