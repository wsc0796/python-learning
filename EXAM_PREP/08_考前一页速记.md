# 考前一页速记

## 必背边界

- `range(a, b)` 不包含 `b`。
- 切片 `s[a:b]` 不包含 `b`。
- `input()` 返回字符串。
- 函数没有 `return` 就返回 `None`。
- `find()` 找不到返回 `-1`。
- `dict[key]` 不存在会报错，`dict.get(key, 默认值)` 不会。
- `w` 会覆盖文件，`a` 是追加。
- 循环 `else` 只有没有 `break` 时才执行。

## 必会结构

- if 分支
- for/range 循环
- while 菜单循环
- 列表增删改查
- 字典计数
- 函数参数和返回值
- class + `__init__` + 方法
- with open 读写文件
- try/except/finally

## 考场先后顺序

1. 先拿基础语法和输出题。
2. 改错题先修语法，再修逻辑。
3. 综合题先写菜单和数据结构，再补函数细节。
4. 不确定的高级写法不用，优先写最朴素的基础写法。
