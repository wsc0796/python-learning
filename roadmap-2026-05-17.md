## 八、最重要的提醒

`★ Insight ─────────────────────────────────────`
- **你现在最大的敌人不是"知识不够"，而是"广度太大导致每个方向都浅尝辄止"。** Python 25 专题 + CS50P 11 讲 + 扣子 14 章 + Hello-Agents 16 章 + 苍穹外卖 + 考研 408——如果不做取舍，你会在每个方向上花 20% 的时间得到 80% 的浅度理解。暑假必须收缩到 3 条线：408 + AI 项目 + Hot100。
- **Practice TODO 清债是第一优先级。** 笔记读了 ≠ 会了。你的 26-unit-tests、27-regex、17-context-iterator-async 三个专题的 practice TODO 是现阶段性价比最高的投入——每个最多 1-2 小时，但能把你从"看过"变成"写过"。
- **苍穹外卖收尾不要追求完美。** 整理一份 2 页的"项目梳理笔记"（架构图 + 核心表结构 + 你负责的模块 + 遇到的技术难点），比把所有代码重构一遍有价值得多。这份笔记是以后面试时你讲项目的提纲。
- **AI-Meeting 不要急着跑代码。** 先读它的 README 和架构图，理解 5 个 Agent 怎么协同面试的。项目是参考，不是作业。理解设计思路 > 跑通代码。
