## 每日流程（固定30分钟）

```
0-5 min    读题，自己想（想不出也正常）
5-10 min   看 PDF 思路部分，不碰代码
10-20 min  自己写代码（C 或 Python）
20-25 min  对比 PDF 代码，看差异
25-30 min  在本文件末尾打标记：✓ / △（思路懂代码没写通）/ ✗（没懂）
```

## 标记体系

完成后在每道题后加标记：
- `✓` 自己写出来了
- `△` 看思路后写出来了
- `✗` 看了答案才懂
- `空白` 还没做

例如：
```
| 1 | S | 1 | 两数之和 ✓ |
| 2 | S | 283 | 移动零 △（双指针方向想反了） |
```

下次复习时优先看 ✗ 和 △。
