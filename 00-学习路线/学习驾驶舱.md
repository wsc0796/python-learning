## 今天怎么用

每天只按这个顺序打开：

1. 本文件：确认今天属于哪条主线。
2. 当前主线文件：只打开一份计划，不同时展开所有路线。
3. 对应章节目录：读 `theory_*.md`，手敲 `practice_*.py`。
4. **卡住时查连接线**：[[知识连接线]]（模块间依赖）、[[课程大纲串联线]]（课程章节因果）
5. `progress/`：收尾时记录错误、复述和明天继续点。

固定学习动作仍然按 [[python学习固定流程]]：

```text
读 -> 敲 -> 破 -> 跑 -> 记 -> 述 -> 回顾
```

## 物理整理原则

现在不建议大规模移动目录，因为这里同时是 Obsidian Vault 和 Git 仓库，移动会影响双链、历史路径和已有学习记录。

推荐策略：

1. 先用本文件和 [[内容分类总索引]] 控制视野。
2. 只清理明显临时文件，例如 `~$*.docx`、`_tmp_*.docx`、`__pycache__`。
3. 等主线稳定后，再考虑把旧练习和课程实验移进 `_archive`。
