# 考前必须背熟的代码模板

## 1. if 模板

```python
if 条件:
    ...
elif 条件:
    ...
else:
    ...
```

## 2. for 模板

```python
for i in range(n):
    ...
```

## 3. 函数模板

```python
def func(a, b):
    result = a + b
    return result
```

## 4. 类模板

```python
class Student:
    def __init__(self, name, score):
        self.name = name
        self.score = score

    def show(self):
        print(self.name, self.score)
```

## 5. 文件模板

```python
with open("data.txt", "r", encoding="utf-8") as f:
    content = f.read()
```

## 6. 异常模板

```python
try:
    num = int(input("请输入数字："))
except ValueError as error:
    print("输入错误", error)
else:
    print(num)
finally:
    print("程序结束")
```
