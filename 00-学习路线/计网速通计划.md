# 计网 7 周复习计划

教材：自顶向下方法（Kurose & Ross）
笔记：Desktop/Shirakoko_Notes/计算机网络（自顶向下方法）课程笔记.pdf
网课时长：24h

---

## 核心策略

- 网课开 **1.5x-2x 倍速**，24h 压缩到 12-16h
- 看完一章立刻做题，不攒到最后
- 计网也是考研 408 内容，现在学透暑假就少一件事

## 每周节奏

| 时段 | 内容 |
|------|------|
| 周二四晚间（深度块1） | 各看一小节网课（~30min，1.5x 速） |
| 周六上午 | 集中看 2-3h 网课推进度 |
| 周日上午 | 做该章的练习题 + 整理错题 |

7 周 × 周末 3h + 平时穿插 ≈ 够用。

## 章节安排

| 周次 | 内容 | 网课时长 | 做题 |
|------|------|---------|------|
| 第 1 周 | 第 1 章：计算机网络概述 | 3h | 课后选择题 |
| 第 2 周 | 第 2 章：应用层（HTTP/DNS/FTP/Email） | 5h | 应用层计算题 |
| 第 3 周 | 第 3 章上：运输层（UDP/RDT/GBN/SR） | 4h | 运输层简答 |
| 第 4 周 | 第 3 章下：TCP 拥塞控制 | 3h | TCP 拥塞控制画图题 |
| 第 5 周 | 第 4 章：网络层（IP/子网划分/路由协议） | 5h | 子网划分 + 路由计算 |
| 第 6 周 | 第 5 章：链路层（CRC/MAC/CSMA/CD） | 4h | 链路层计算 |
| 第 7 周 | 总复习 + 真题模拟 | — | 整套试卷 |

## 优先级

如果你中间某周时间不够，按这个顺序砍：

TCP 拥塞控制 > 子网划分/CIDR > 路由算法(LS/DV) > HTTP/DNS > GBN/SR > CSMA/CD > CRC 计算

前 3 项通常占考试 60% 以上。

## Day 1：应用层 + 运输层上半（3h）

| 时间 | 内容 | 重点 |
|------|------|------|
| 0:00-0:45 | 应用层：HTTP / FTP / Email / DNS | 非持久 vs 持久连接、状态码、DNS 解析流程、HTTP 方法 |
| 0:45-1:00 | Socket 编程 | TCP Socket / UDP Socket 流程区别（握手 vs 无连接） |
| 1:00-1:10 | 休息 | |
| 1:10-2:00 | 运输层：UDP + RDT 协议 | RDT 1.0→2.0→2.1→2.2→3.0 演进逻辑（为什么需要 seq/ACK/NAK/timer） |
| 2:00-2:45 | 流水线协议：GBN vs SR | 窗口大小、ACK 机制、超时重传的区别 |
| 2:45-3:00 | 回顾 Day 1 知识点 | |

**Day 1 必会考点：**
- HTTP 持久/非持久连接的 RTT 计算
- DNS 递归 vs 迭代查询
- DNS 资源记录类型（A/NS/CNAME/MX）
- RDT 协议演变中各版本解决了什么问题
- GBN 与 SR 窗口大小限制（N-1 vs N/2）

| 时间 | 内容 | 重点 |

## Day 3：链路层 + 查漏补缺（3h）

| 时间 | 内容 | 重点 |

## 各层速记表

| 层 | PDU | 设备 | 代表性协议 |
|----|-----|------|-----------|
| 应用层 | Message | 应用 | HTTP / FTP / SMTP / DNS |
| 运输层 | Segment / Datagram | 无 | TCP / UDP |
| 网络层 | Packet / Datagram | 路由器 | IP / ICMP / OSPF / BGP |
| 链路层 | Frame | 交换机 / 网桥 | Ethernet / ARP |
| 物理层 | Bit | 中继器 / 集线器 | — |
