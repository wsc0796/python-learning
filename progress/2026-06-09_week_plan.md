# 2026-06-09 到 2026-06-14 加速周计划

本周主题：Python 基础回炉 + FastAPI 分层实战 + 后端框架入门 + Agent 架构迁移。

结论：后端框架部分应该现在开始，但不要另开一条 Java/Spring 重线。当前最优路线是：

```text
Python 基础补洞 -> FastAPI 分层 CRUD -> Spring Boot 概念对照 -> AI-Meeting/Agent 项目迁移
```

也就是说：用 FastAPI 把 Controller / DTO / Service / Repository / DI / Test 这些框架能力练出来，再用 Spring Boot 对照表理解 Java 项目。

## 是否开始后端框架学习？

答案：开始，但只开始“框架结构”，不要开始“框架全家桶”。

现在该学：

- HTTP route / Controller
- DTO / VO / Pydantic / `@Valid`
- Service / Repository 分层
- 依赖注入
- 异常转换
- 测试和接口验证

现在不该学：

- Spring Security
- 微服务
- 消息队列
- 分布式事务
- 高并发 Redis 细节
- 复杂 ORM 配置

判断标准：凡是能帮助你读懂 AI-Meeting 的 Controller、Service、Mapper，就学；凡是让你离当前项目链路更远，就先放。
